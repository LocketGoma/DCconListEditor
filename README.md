BBCC Addon : DCCon List Editor
===
version 0.0.5

## 개요
해당 프로젝트는 트위치 확장 프로그램인 Bridge BBCC (https://github.com/krynen/BridgeBBCC) 프로젝트의 애드온으로 제작되는 프로젝트입니다.

현재 개발중.


## 용어 정리
1. 디시콘 리스트 : '실제 디시콘 이미지 파일'들의 목록을 의미.
2. 디시콘 엔트리 : 'dccon_list.js' 파일에 들어가 있는 목록을 의미.
